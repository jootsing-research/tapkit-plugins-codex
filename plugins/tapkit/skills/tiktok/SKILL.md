# TikTok

TikTok is heavily full-screen and motion-driven. Always start from a screenshot, verify after each interaction, and expect UI chrome to hide until you tap or switch views.

Use the `tapkit` skill's clipboard workflow for comments and search.

## Main Tabs

- `Home`
- `Friends`
- `Create`
- `Inbox`
- `Profile`

## Common Tasks

### Browse Videos

1. Open `Home`.
2. Swipe up to move to the next video.
3. Use the right-side action rail for like, comments, share, and profile jumps.

### Comment on a Video

1. Tap the comment icon.
2. Wait for the comments sheet.
3. Activate the text field.
4. Paste the comment and send it.

### Search

1. Open search from the top-right or from another visible entry point.
2. Activate the field.
3. Paste the query and submit.
4. Open the matching creator, sound, or hashtag.

## Gotchas

- The visible controls can change between For You, Following, live streams, and profile screens.
- Comments, share panels, and creator sheets stack on top of the feed.
- A missed tap on TikTok often advances content, so verify quickly after each action.
