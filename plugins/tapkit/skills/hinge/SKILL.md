# Hinge

Open Hinge directly if possible. Hinge is highly visual, so take a screenshot before each action and verify after every like, pass, or message.

Use the `tapkit` skill's clipboard-based text flow whenever Hinge asks for a comment or message.

## Main Tabs

- `Discover`
- `Standouts`
- `Likes You`
- `Matches`
- `Profile`

## Common Tasks

### Like or Pass on a Profile

1. Start on `Discover`.
2. Scroll through the current profile.
3. Tap the like control on a specific prompt or photo, or choose `X` to pass.
4. Verify that the next profile loaded.

### Send a Like with a Comment

1. Tap the part of the profile you want to respond to.
2. Open the comment field if Hinge shows one.
3. Paste the message.
4. Send the like.

### Message a Match

1. Open `Matches`.
2. Tap the target conversation.
3. Paste the message into the input field.
4. Tap the send button and verify the new message bubble appears.

## Gotchas

- The bottom tab bar may disappear once you are deep in a profile or thread.
- Some actions consume limited likes or roses, so read the screen carefully before confirming.
- Hinge often uses small icon-only buttons. Screenshot and zoom mentally before tapping.
