# X / Twitter

Open X and re-screenshot often. The app mixes full-screen timelines, overlays, and a floating compose button, so confirm the current surface before tapping.

Use the `tapkit` skill's clipboard workflow for replies, posts, search, and DMs.

## Main Navigation

Common bottom tabs include:

- `Home`
- `Search`
- `Grok` or another AI tab
- `Notifications`
- `Messages`

There is often a floating compose button in the lower-right area.

## Common Tasks

### Browse the Feed

1. Open `Home`.
2. Scroll with short upward swipes.
3. Distinguish between `For You` and `Following` if both tabs are visible.

### Like or Reply

1. Open the post detail if necessary.
2. Tap the heart icon to like.
3. Tap the reply icon.
4. Paste the reply text and submit.

### Compose a Post

1. Tap the floating compose button.
2. Activate the text field.
3. Paste the post text.
4. Tap `Post`.

### Search

1. Open `Search`.
2. Paste the query into the search field.
3. Open the matching topic, person, or post.

## Gotchas

- A floating compose button can overlap content and attract accidental taps.
- Timelines refresh aggressively after navigation.
- X frequently hides extra actions behind overflow menus or a press-and-hold gesture.
