# Telegram

Open Telegram and work from screenshots. Telegram exposes many actions through icons in the top bar, so inspect both the top and bottom of the screen before acting.

Use the `tapkit` skill's clipboard workflow for searches and outbound messages.

## Main Tabs

- `Contacts`
- `Calls`
- `Chats`
- `Settings`
- `Search`

`Chats` is usually the main working area.

## Common Tasks

### Read or Send a Message

1. Open `Chats`.
2. Tap the target conversation.
3. Review the thread.
4. Paste a new message into the input field and send it.

### Search for a Chat or Message

1. Open `Search` or the visible search control.
2. Paste the query.
3. Open the matching chat, contact, or message result.

### Interact with a Group or Channel

1. Open the chat.
2. Read pinned messages, recent posts, or the member list as needed.
3. Use the message composer only if the chat allows sending.

## Gotchas

- Telegram uses icon-only send, attach, and voice-message buttons.
- Long-pressing chats often opens useful context menus.
- Search can return people, chats, channels, and message hits in the same view, so read carefully before tapping.
