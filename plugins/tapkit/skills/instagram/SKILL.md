# Instagram

Open Instagram and work from screenshots. The interface uses icon-heavy navigation, so rely on visible icons and context rather than fixed positions.

Use the `tapkit` skill's clipboard workflow for comments, search, and DMs.

## Main Navigation

Instagram commonly exposes:

- `Home`
- `Reels`
- `Messages` or `DMs`
- `Search` or `Explore`
- `Profile`

The exact order shifts across versions.

## Common Tasks

### Browse Feed or Reels

1. Open `Home` or `Reels`.
2. Swipe up to move through content.
3. Use visible heart, comment, or share icons for interactions.

### Like or Comment on a Post

1. Open the post details if needed.
2. Tap the heart icon to like.
3. Tap the comment control.
4. Paste the comment text and submit.

### Search for an Account or Topic

1. Open `Search` or `Explore`.
2. Activate the search field.
3. Paste the query and submit it.
4. Open the matching result.

### Send a DM

1. Open `Messages`.
2. Search for the recipient if necessary.
3. Open the thread.
4. Paste the message and send it.

## Gotchas

- Full-screen media views and comment sheets can hide the main navigation.
- Story rings, DM badges, and overlays are visually dense. Re-screenshot often.
- Instagram frequently starts on a dark UI, which can make tappable hit targets harder to judge.
