# LinkedIn

Open LinkedIn and verify each screen with a screenshot. LinkedIn uses both bottom navigation and top-bar actions, so check both areas before deciding where to tap.

Use the `tapkit` skill's clipboard workflow for comments, searches, posts, and messages.

## Main Navigation

Common LinkedIn tabs include:

- `Home`
- `Video`
- `My Network`
- `Notifications`
- `Jobs`

Search, messages, and profile actions are often exposed from the top bar.

## Common Tasks

### Browse the Feed

1. Open `Home`.
2. Scroll through posts.
3. Use the like, comment, or repost controls on the target card.

### Connect with Someone

1. Use search from the top bar.
2. Paste the person's name.
3. Open the profile.
4. Tap `Connect`, `Follow`, or the overflow menu as appropriate.

### Search Jobs

1. Open `Jobs`.
2. Use the search field.
3. Paste the title, company, or location query.
4. Open the listing and inspect details.

### Send a Message

1. Open the messages or chat entry point.
2. Select the conversation.
3. Paste the message and send it.

## Gotchas

- Messages may live behind a top-bar icon instead of a dedicated bottom tab.
- Job-search filters often open sheets that require an extra confirmation tap.
- LinkedIn sometimes opens composers or drawers that partially cover the underlying screen.
